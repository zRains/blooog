Axios 详解

因为需求在主键挂载后发出两个 API 请求，如果分开写会导致主键读取不到数据。于是看了一下 Axios 的并发，但发现有点不太一样。

Cookies 登录验证

很少写登录方面的东西，不过这东西以后肯定是要做的。自己写了个后台就用上了登录，可能有点 low 但还是学到了不少。

editor 图片上传示例

关于博客的展示组件自己选了好久，另一个mavonEditor也不错，但支持的插件不多，于是选了这个 vue-markdown-editor（均国产✌️），支持自定义高亮语法包和插件自定义，最重要的是包小啊。研究了一下图片上传的功能，挺香的。官方文档有实列但过于简洁。记录一下编写过程。

ubuntu搭建node环境

闲来无事在自己的后端服务器移到自己的树莓派上，实际上是我的云服务器快过期了💦，在此记录搭建过程，其实大部分是关于如何安装最新版 node 的。网上大部分教程都是下载源码直接编译，但我一想到前些日子手动编译安装的 Nginx 想了想还是算了。
