在把网站部署后发现 static 文件里面的东西访问不了，因为文件里只有一个favicon.png文件，通过控制台发现是通过.../_nuxt/favicon.png访问的，照葫芦画瓢发现还是访问不到。一直以为问题是 Nuxt 没有配置好，到头来还是出在反向代理上啊。

☀️ 解决

在 Nginx 配置文件里添加如下匹配规则：

# 如果需要其它类型文件，在()里继续添加后缀即可
location ~.*\.(css|js|jpg) {
	# zy注意替换为自己的端口号。需要注意两点：1.proxy_pass的值不能有 / ，2.也不能有uri
	proxy_pass http://127.0.0.1:8080;
}
